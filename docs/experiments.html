閱讀方式

主要支援

對應 supported path，代表主要路線已跑通，結果與預期 matrix 一致。

舊路徑

對應 deprecated path，代表功能存在，但屬於舊路徑或 lab-only 驗證，不應直接視為 forward-looking support。

產品限制

對應 limitation，代表不是 VM 起不來，而是目前沒有建立出可信的產品路徑或官方支援模型。
